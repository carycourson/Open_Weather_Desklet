# Open Weather Desklet

A lightweight, modern weather desklet for the Cinnamon Desktop Environment. 

Built from scratch to demonstrate efficient asynchronous networking in GJS (Gnome JavaScript) without legacy dependencies.

## Key Features
- **Engine:** Powered by OpenWeatherMap OneCall 3.0.
- **Performance:** Non-blocking async HTTP calls using LibSoup 3.
- **Design:** Clean UI supporting scalable VClouds icon sets.
